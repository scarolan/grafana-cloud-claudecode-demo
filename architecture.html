Hook Event Lifecycle
SessionStart
Generate trace_id (128-bit) root_span_id (64-bit) Save session.json
UserPromptSubmit
Close prev prompt span (if any) Generate prompt_span_id Save current_prompt
PreToolUse
Record start_ns tool_use_id Append events.jsonl
PostToolUse
Load session + start_ns Emit span parent = prompt_span_id attrs: tool.name, user.prompt…
SessionEnd
Close last prompt span Emit root session span via FixedIdGenerator Cleanup temp files
